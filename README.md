# Incident Management App

A full-stack incident management application built with React on the frontend, ASP.NET Core on the backend, and hosted on Microsoft Azure. The application allows users to report,
track, and manage incidents in a secure and scalable cloud environment.

---

## Tech Stack

### Frontend
- React
- JavaScript
- API communication
- HTML5 and CSS

### Backend
- ASP.NET Core Web API
- C#
- Entity Framework Core
- RESTful API architecture

### Cloud Infrastructure
- Microsoft Azure
  - Azure App Service
  - Azure SQL Database
  - Azure Storage
  - Azure Application Insights

---

## Features

- Create, update, and manage incidents
- RESTful API for incident operations
- Secure backend with role-based access
- Scalable cloud hosting on Azure
- Responsive user interface
- Centralized error handling and logging
